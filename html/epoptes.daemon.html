class epoptes.daemon.bashplex.DelimitedBashReceiver[source]¶

Bases: twisted.internet.protocol.Protocol

Send bash commands followed by ” echo <delimiter>”.

Buffer responses until the given delimiter is found.
check_for_further_responses()[source]¶

See if there are more responses in the buffer. The theory here is that if we got one already, we have less than one packet of data left in the buffer, so reading it all isn’t a big deal.

command(cmd, delimiter=None)[source]¶

Send a command and return its associated Deferred. We don’t do any state-checking here. If another command is in progress, it’ll probably be okay, but if serialization is required, it should be done elsewhere.
