epoptes.common.config.expand_filename(filename)[source]¶

Return the full path for the specified user settings file.

Ensure that the directory where filename resides, exists.

epoptes.common.config.read_groups(filename)[source]¶

Parse a JSON file and create the appropriate group and client objects. Return a 2-tuple with a client instances list and a group instances list.

epoptes.common.config.read_ini_file(filename)[source]¶

Return a ConfigParser from the contents of a configuration file.

epoptes.common.config.read_plain_file(filename)[source]¶

Return the whole contents of a plain text file into a string list. If the file doesn’t exist or isn’t readable, return an empty list.

epoptes.common.config.read_shell_file(filename)[source]¶

Return the variables of a shell-like configuration file in a dict. If the file doesn’t exist or isn’t readable, return an empty list. Also strip all comments, if any.

epoptes.common.config.save_groups(filename, model)[source]¶

Save the groups and their members from model (gtk.ListStore) in JSON format.
