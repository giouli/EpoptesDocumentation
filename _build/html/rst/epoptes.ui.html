epoptes.ui.common module¶

Define required gi package versions in a common place, and install gettext.

Rationale: gi requires something like:

import gi gi.require_version(‘Gtk’, ‘3.0’) from gi.repository import Gtk

This conflicts with https://www.python.org/dev/peps/pep-0008/#imports and triggers pycodestyle’s “E402 module level import not at top of file”. The following is a bit better:

import sys # Import standard library modules

import twisted # Import third party modules

from epoptes.ui.common import gettext as _ # Import local modules from gi.repository import Gtk, Gdk

That last line “only” triggers pylint’s “wrong-import-position” once.

epoptes.ui.notifications module¶

Notifications. Keep at most 10 lines of headings and messages. A new bubble is created after 10 secs of inactivity. That means that the oldest event may show for up to 9*10 seconds or so.

Example bubble layout: [icon] Epoptes Connected:

user1 on host1 user2 on host2
Disconnected:
user3 from host3
Shut down:
host4
Connected:
user5 on host5

shows up to only 6 lines when hovered, merges ‘
